Public Whiteboard
The Public Whiteboard is a globally accessible platform that allows internet users to collaborate in real time on a shared virtual whiteboard. Any contributions made are visible to everyone online, fostering a dynamic and interactive public space for creativity and expression.
Cangjie Trainer
The Cangjie Trainer is a practice tool for learning and improving Cangjie typing skills. It allows users to customize the difficulty level, enabling them to tailor their practice sessions to match their current proficiency and learning goals.
Cangjie Dictionary
The Cangjie Dictionary is a tool that supports simultaneous lookup of multiple words using the Cangjie input method, making it efficient for users to find character codes quickly and accurately.
Personal Blog
